# README

## Cloud9 commands for develop env

use this command in bash tab.

run database (postgresql):
`sudo service postgresql start`

run project:
`rails s -p $PORT -b $IP`

install deps:
`bundle install`

run migration:
`rake db:migrate`

run seeds:
`rake db:seed`

For run project:

1. run db
2. run project
3. open link
